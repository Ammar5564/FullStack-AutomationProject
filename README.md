# Automation Testing Project using Selenium (Java) + Backend Login Feature

A demo project combining *Backend Development* ,*Frontend Development* and *Automation Testing*.  
Implemented a real backend login API and an end-to-end *Selenium Test Automation framework* using Java & TestNG.

---

## Project Goal

- Learn and implement *Automation Testing* from scratch
- Apply *Page Object Model* for better test maintainability
- Connect *frontend login tests* with a *real backend API*
- Showcase clean *project structure* for testing and backend development

---

## Technologies Used

- *Java* for backend and testing
- *Spring Boot* for backend API
- *jpa* for the DataBase
- *Selenium WebDriver* for browser automation
- *TestNG* for test management
- *Maven* for project build and dependencies
- *Git & GitHub* for version control
